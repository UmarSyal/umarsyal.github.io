Marketing Company Sales Software

Sales dashboard for a real marketing company.

Features:

  • Authentication and Authorization
  • 3 Types of Users(Admin, Stockist, Customer)
  • Admin can add Stockist and issue stock
  • Admin can view all Stockists and Customers
  • Stockist can make a booking
  • If a customer is a first timer, their account gets created automatically
  • Stockist can deliver a booked product
  • Commissions get distributed to Stockists and Customers on a booking
  • Stockist can view only their own customers
  • Customers can view only their own purchase history
  • Booking slips in PDF format
  • All the user dashboards have all the relevant data for quick access

Tools:

  • Python
  • Django
  • PostgreSQL
  • HTML5
  • CSS3
  • Bootstrap3
  • Heroku
